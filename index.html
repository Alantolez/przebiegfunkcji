Przebieg zmienności funkcji

Funkcja:

\[f(x) = \frac{x^2 + 6x + 9}{x^2 - 3}\]

1. Dziedzina funkcji

\[f(x) =x^2 - 3\neq 0\] \[x^2 \neq 3\] \[D_{f} : x\in \mathbb{R} \smallsetminus \left\{-\sqrt{3}, \sqrt{3} \right\}\]

2. Miejsce zerowe

\[f(x) = 0\] \[\frac{x^2 + 6x + 9}{x^2 - 3} = 0\] \[(x+3)^2 = 0 \] \[x = -3 \]

3. Punkt przecięcia z OY

\[f(0) = \frac{0^2 + 0 + 9}{0^2 - 3}\] \[f(0) = -3\] \[P_{pOY}:(0,-3)\]

4. Parzystość/nieparzystość

\[D_{f} : x\in \mathbb{R} \smallsetminus \left\{-\sqrt{3}, \sqrt{3} \right\}\] \[f(-x) = \frac{(-x)^2 - 6x + 9}{(-x)^2 - 3} \] \[f(-x) = \frac{x^2 - 6x + 9}{x^2 - 3} \neq f(x) \]

Funkcja nie jest parzysta.

\[-f(x) = \frac{-x^2 - 6x - 9}{-x^2 + 3} \neq f(x)\]

Funkcja nie jest nieparzysta.

5. Granice

\[\lim_{x \to \pm \infty} f(x) = 1\] \[\lim_{x \to -\sqrt3^{-}} f(x) = \infty\] \[\lim_{x \to -\sqrt3^{+}} f(x) = -\infty\] \[\lim_{x \to \sqrt3^{-}} f(x) = -\infty\] \[\lim_{x \to \sqrt3^{+}} f(x) = \infty\]

6. Asymptoty

\[\text{Pozioma:}\ y = 1\] \[\text{Pionowa:}\ x = -\sqrt3, x = \sqrt3\] \[\text{Ukośna:}\ brak\]

7. Monotoniczność:

\[{f}'(x) = \frac{ ( 2x+6 )( x^{2}-3 )-2x(x^2 + 6x + 9)}{(x^{2}-3)^2} \]

Po przekształceniach...

\[{f}'(x) = \frac{6(-x^2 - 4x -3)}{(x^{2}-3)^2} \] \[D_{f} = D_{{f}'}\] \[\forall_{x\in D_{{f}'}} ( x^{2}-3 )^{2}> 0\]

Znak pochodnej zależy od licznika.

\[-x^{2}-4x-3=0\] \[-(x+1)(x+3) = 0\] \[x = -1 \vee x = -3\]
\[f\nearrow < -3, -1>\] \[f\searrow (-\infty, -3>, < -1, \infty)\]

8. Ekstrema:

\[\text{Dla}\ x = -3,\ y_{min}=0\] \[\text{Dla} x=-1,\ y_{max}=-2\]
